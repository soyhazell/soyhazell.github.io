What is this repository?

This repository includes a folder and article.html file for students in a web design class to fork and fix.

About the Article

The content of the article is from The Conversation, a blog put out by academics under Creative Commons licenses - non-derivative.

It is very important that you do not change the actual article content! This will break the copyright license agreement. See the Conversation republishing guideline webpage for more information.

How to Use This Repository

  • Fork this repository
  • Edit the article.html file
  • Submit the link to your repository to the Canvas assignment

Learning Objectives:

  • Use semantic HTML elements to give a web page meaning
  • Use an HTML validator service and fix errors in code based on web standards
